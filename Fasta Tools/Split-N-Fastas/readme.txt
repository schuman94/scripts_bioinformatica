Este script es una herramienta interactiva diseñada para facilitar la manipulación de archivos en formato FASTA, utilizados comúnmente para almacenar secuencias biológicas.
Permite dividir un archivo FASTA grande en múltiples archivos más pequeños, cada uno conteniendo un número específico de secuencias.

Archivo de Entrada: Un archivo FASTA que contiene múltiples registros de secuencias. Debe ser un archivo con extensión .fasta o .fa.
Directorio de Salida: La ubicación donde se guardarán los archivos FASTA resultantes.
Número de Secuencias por Archivo: Este parámetro define cuántas secuencias contiene cada archivo FASTA resultante.

Se generan varios archivos FASTA, cada uno conteniendo el número especificado de secuencias.
Los archivos son nombrados automáticamente basándose en el nombre del archivo original con un sufijo numérico que indica su secuencia en el conjunto dividido.
