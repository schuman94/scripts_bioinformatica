Este programa es una herramienta interactiva diseñada para separar las secuencias de un fichero fasta en ficheros individuales.

En fasta file se debe seleccionar el fichero fasta que contiene todas las secuencias.
En oupt directory se debe seleccionar el directorio donde se crearán todos los ficheros fasta individuales.

Cada fichero fasta generado contendrá una única secuencia. El nombre del fichero será el ID de la secuencia que contiene.
