Este programa está diseñado para facilitar el registro del promedio de valores PLDDT, los cuales se tratan de indicadores de la precisión de las estructuras de proteínas predichas computacionalmente mediate Alphafold.
El usuario puede seleccionar fácilmente las carpetas de entrada, donde se encuentran los resultados de las diferentes proteínas, y salida para almacenar los datos, respectivamente.

Cómo Usar el Programa
Inicio del Programa: Al iniciar el programa, se abrirá una ventana con opciones para seleccionar las carpetas y ejecutar el análisis.
Seleccionar Carpeta de Entrada: Utiliza el botón de "Buscar" en la sección "Carpeta de entrada" para seleccionar la carpeta donde se encuentran los archivos JSON de los cuales se extraerán los datos PLDDT.
Seleccionar Carpeta de Salida: Utiliza el botón de "Buscar" en la sección "Carpeta de salida" para elegir la ubicación donde deseas guardar el archivo Excel generado con los resultados del análisis.
Ejecutar el Análisis: Una vez seleccionadas ambas carpetas, presiona el botón "Ejecutar" para comenzar el proceso de análisis. El programa verificará que se hayan seleccionado las carpetas correctamente y luego procederá a analizar los archivos especificados.
Resultados: Al finalizar el análisis, el programa generará un archivo Excel llamado registro_plddt.xlsx en la carpeta de salida seleccionada. Este archivo contiene una lista de proteínas analizadas junto con el promedio de sus valores PLDDT.
Resultados de Salida
El archivo Excel generado (registro_plddt.xlsx) incluirá las siguientes columnas:

Proteina: Nombre de la proteína analizada.
Media PLDDT: Promedio de los valores PLDDT calculados para la proteína.
